Plugins / Tools

This theme includes many freely available jQuery plugins and tools to make adding advanced features to the site quick and easy.

Lightbox / Gallery

The jQuery lightBox plugin is set up in this template and ready to go. Although it can be configured in any way, by default it will turn any link with a rel="lightbox" into a lightbox.

<a href="large-image.jpg" rel="lightbox" title="Optional Title"><img src="thumb.jpg" alt="image" /></a>

Image/HTML Slider

This template comes with jCarousel Lite set up and can be reused for any size of image and content on your site.

The sliders above are in a grid_12 and grid_4 widths but can fit any space, be sure to read the jCarousel Lite setup and documentation.

  • HTML Content

    All sides must be the same width/height, which is taken care of in images with height="" and width="" tags.

    When using HTML content, I simply add a height/width style to the LI elements in the slider.

    
<li class="item-1" style="width:940px; height:300px;">
	<div style="padding:20px;">
		<h3>HTML Content</h3>
	</div>
</li>
<li class="item-2"><img src="images/940x300.gif" alt="..." width="940" height="300" /></li>
<!-- etc... -->

Drop Down Menu

This template is using the jQuery Superfish v1.4.8 drop down menu to allow multiple levels of dropdown navigation. The delay, animation, speed and other settings are fully adjustable through the js/general.js file.

The menu has been tested up to three levels deep and also have added images, see Contact Us > Locations which opens a larger image in a lightbox. The styles and colors of the menu are edited through the standard CSS file. Just edit anything with a .sf-menu class.

Scroll to Top

Allow your users to smoothly scroll to the top of a page by just adding class="scroll-top" to any link and it will bring the users back to the top of the current page.
